# 姫路城 大天守ガイド（リニューアル） — シングルページ

## 概要
大天守 地階〜6階の見どころをわかりやすく表示する観光ガイドのフロントエンド（静的）。モバイル優先で、フロア選択→詳細→フロアマップという体験を想定。

## 使い方
1. `index.html` をブラウザで開くだけ。
2. 左（スマホは上）でフロアを選ぶと内容が切替わる。
3. `フロアマップを見る`でモーダル表示（運用時は実画像に差し替え）。

## 特長
- 写真を大きく見せるヒーロー構成
- 所要時間・見どころを即表示
- キーボード操作（上下・Enter・数字キー）に対応
- 公式情報へのリンクを用意（最新情報は必ず公式で確認を）。:contentReference[oaicite:2]{index=2}

## 今後の改善案（優先度付き）
1. 各フロアの高解像度写真に差し替え（撮影: 窓越し、床材のディテールなど） — 優先度: 高  
2. フロアマップにホットスポット（写真スポット）を配置してピン連動 — 優先度: 高  
3. 英語・中国語など多言語対応（i18n仕組み） — 優先度: 中  
4. AR/360ビューの導入（最上階の360°ビュー） — 優先度: 低  
5. 屋内案内（バリアフリー情報、段差・車椅子ルート）を明確化 — 優先度: 高
